Hosted here: [https://draw-together-y3bz.onrender.com/]([url](https://draw-together-y3bz.onrender.com/))

Draw uses html, css, javascript, and node.js to host a real-time whiteboard that anyone on the internet can interact with! 
Drawings will appear in real time on other users screens as they are made.
It serves as a communal whiteboard for the internet, with features like a pencil and eraser, different line widths, and changable colors.
